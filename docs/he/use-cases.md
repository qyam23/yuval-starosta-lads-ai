# מקרי שימוש

## 1. שכבת נראות תעשייתית לקו קיים
הטמעת שכבת נראות קלה מעל מידע קיים במפעל כדי לארגן מצבי מכונה, אותות ייצור והקשר תפעולי.

## 2. ייצוב קו אקסטרוזיה
בחינת תנאי תהליך, התנהגות בקרה ואינטראקציה בין ציוד כדי לצמצם חוסר יציבות ולשפר חזרתיות.

## 3. תכנון מפעל לפי לוגיקת תהליך
תכנון מבנה מפעל, זרימת חומר, utilities ותנועת מפעילים מתוך התהליך פנימה, ולא מתוך אילוצי מעטפת בלבד.

## 4. מודרניזציית בקרה עם מוכנות לנתונים
עדכון PLC, HMI, אזעקות ומבנה האותות כך שהקו יהיה קל יותר לתפעול היום וקל יותר לניתוח מחר.

## ערך תפעולי משותף
בכל מקרי השימוש האלה, הערך המשותף הוא מבנה טוב יותר, תפעול יציב יותר, אבחון מהיר יותר וקבלת החלטות הנדסית מדויקת יותר.
